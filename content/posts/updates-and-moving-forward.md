Hello

The site may look a little different and that is because I rebuilt it and changed how it is hosted.

For a while now I have wanted to get away from WordPress and change to anything else. I preferred something simple and I think I have found a solution. 

Navigating WordPress is incredibly annoying. And when you login to the admin dashboard, you get bombarded with stuff that needs updated. Plugins need updated. Themes need updated. It gets aggravating when you just want to post something quick.

Also the interface for building your site on Wordpress is not the easiest to navigate. I'm sure it is for those that spend a lot of time in it but for a tech blog, it is overkill and not necessary.

So what did I change to? 
 
Hugo and GitHub pages.

This will be my first post after the migration so we will see how it goes but I can save this as a markdown file and put it in the correct directory on my computer and run git add and git push and wait for it to finish and it is up.

I made a few visual changes to the appearance of the site last night and made a few commits trying to get the hang of it.

I hope this system will be much easier and less time intensive without all of the updates and allow me to publish a lot more frequently.

I can write markdown on my phone and save the file to Google Drive so when I get to my computer, I can push it and if all goes as planned, it is live.

That's the plan anyway.

Until next time.
